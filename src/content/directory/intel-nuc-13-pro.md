## Overview

The Intel NUC 13 Pro is a tiny x86 PC that can run Docker, agents, and small models continuously. Pair it with an external GPU for heavier inference.

## Specs

| Spec | Value |
|---|---|
| CPU | Up to Intel Core i7-1370P |
| RAM | Up to 64GB DDR4/DDR5 |
| Storage | M.2 NVMe |
| TDP | 28-64W |

## When to choose

- You need a 24/7 low-power host
- You run agents, bots, or lightweight services
- You may add an external GPU later
