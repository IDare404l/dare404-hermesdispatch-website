## 📦 The Harness: Beelink SER7 Pro

**We run three of these in our stack** — one for IGOR orchestration, one for local inference, one for testing.

| Spec | Result |
|------|--------|
| CPU | AMD Ryzen 7 7840HS (8C/16T) |
| RAM | 32GB DDR5-5600 |
| Ollama Score | **42 tok/s** on llama3.2 3B Q4_K_M |
| Power Draw | 65W under load |
| Price | $649 |

**Why this matters:** At 42 tok/s, you can run a local coding agent in real-time. No cloud costs. No API keys. Just electricity.

**Boise note:** Idaho Power charges ~$0.09/kWh. Running this 24/7 costs about $4.25/month in power. Compare that to GPT-4 API bills.

**Limitations:** Don't expect reasoning beyond middle-school level. For complex tasks, you'll need a 7B+ model — which this box struggles with.
